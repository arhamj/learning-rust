# Learning Rust

## Installation

### MacOS

```shell
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
xcode-select --install
rustup update

# (Or)

brew install rustup
rustup-init
rustc --version
rustup update
```
